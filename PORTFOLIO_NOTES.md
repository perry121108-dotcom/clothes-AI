# clothes AI 作品集說明

Repo：`D:\clothes AI`
GitHub：`https://github.com/perry121108-dotcom/clothes-AI`

## 專案定位

`clothes AI` 是作品集中主打 AI 自動化工作流的展示專案：自動生成穿搭圖片，再透過 Telegram 交付結果。

## 專案方向補充

這個專案最初的出發點其實是「短影音自動化生成」。

後來因為影片 API 成本較高，為了成本控管，展示方向調整為：

- 以圖片生成為主
- 影片作為曾經做過的功能證據保留

這個敘事對 GitHub 很有幫助，因為它能說明：

- 你不是沒做過影片
- 而是做過後，基於成本與可持續性，主動調整為圖片模式

## 素材策略

雲端硬碟裡的 `Clothes ai` 資料夾已有圖片與影片素材，GitHub 展示應整理回 repo：

- `output/`：原始每日輸出
- `assets/github/generated/`：挑選後的代表圖片
- `assets/github/telegram/`：Telegram 接收畫面截圖
- `assets/github/`：GitHub 展示用說明素材
